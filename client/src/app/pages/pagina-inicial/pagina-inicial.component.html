Loan-ely: Framework de Gerência de Empréstimos

Bem-vindo ao Loan-ely, um framework de gerência de empréstimos de produtos, que combina a potência do Spring Boot e a usabilidade do Angular. Com o Loan-ely, estamos redefinindo o modo como os empréstimos funcionam e interagem com seus clientes.

O que é o Loan-ely?

O Loan-ely é um framework solução abrangente criado para modernizar a gestão de empréstimos. Foi um framework desenvolvido por um aluno para a disciplina de Projeto Detalhado de Software no Instituto Metrópole Digital.
